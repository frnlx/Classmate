# Installation

Requires Nodejs

1. `cd code`
2. `npm i`

# Running

Requires setting up .env

Required variables:

- DATABASE_URL
- SUPABASE_ACCESS_TOKEN
- SUPABASE_DB_PASSWORD
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- GOOGLE_CLIENT_ID
- GOOGLE_CLIENT_SECRET
- NEXTAUTH_SECRET
- NEXTAUTH_URL
